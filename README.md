### Running Locally

To run this project on your local machine:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/vyankat70war/portfolio.git
    cd portfolio
    ```

2.  **Install dependencies**
    ```bash
    pnpm install
    # or
    npm install
    ```

3.  **Start the development server**
    ```bash
    pnpm dev
    # or
    npm run dev
    ```

4.  **Build for production**
    ```bash
    pnpm build
    ```
